Turn.js

Muchos conocemos el famoso efecto de revistas interactivas que se presentan desde hace tiempo en la web realizadas con Flash que nos permitían arrastrar una hoja para hacer el cambio y ver las paginas siguientes, un efecto bastante llamativo que todos queríamos usar en alguna ocasión. Ahora que existe HTML5 con todas estas nuevas tecnologías, quizás este efecto sea adoptado. Turn.js es un plugin de JQuery que nos permite realizar el efecto Page-Flip (Tirón de páginas) totalmente con HTML5 y Javascript sin necesidad de algún elemento adicional, tal y como pasaba en Flash. Este famoso efecto está bastante pulido y en el caso del sitio web del mismo plugin, se puede ver que trabaja muy bien.

# Ventaja
